# Figma Make scaffold (archived)

This project was originally generated with Figma Make. These two files are
leftover scaffolding from that tool and are **not used** anywhere in the
actual app:

- `default_shadcn_theme.css` — the default shadcn/ui theme Figma Make
  generates by default. Schedule Beacon's real theme/branding lives in
  `src/styles/theme.css`, which overrides this entirely.
- `Guidelines.md` — an empty template for AI-generation rules, never filled in.

Kept here for reference only. Safe to delete if you don't need them.
